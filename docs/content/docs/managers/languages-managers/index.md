[Java Manager](./java-manager.md)

:   Provides Java installations for test execution, enabling tests to run Java applications with specific versions and configurations.

[Java Ubuntu Manager](./java-ubuntu-manager.md)

:   Provides Java installations specifically for Ubuntu Linux systems, enabling tests to provision and use Java on Ubuntu images.

[Java Windows Manager](./java-windows-manager.md)

:   Provides Java installations specifically for Windows systems, enabling tests to provision and use Java on Windows images.
